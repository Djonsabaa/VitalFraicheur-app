# VitalFra-cheur-app
Cette application mobile offre une interface intuitive et permet aux clients de consulter les produits aquaponies disponibles, d'ajouter des articles à leur panier, de passer des commandes et d'effectuer des paiements en ligne. Elle offre également la possibilité de suivre la livraison en temps réel grâce à un système de géolocalisation intégré.
